# BookStayPro – Hotel Management System

A Full Stack Hotel Booking and Management System developed using Node.js, Express.js, MySQL, HTML, CSS, and JavaScript.

## Features
- User Registration & Login
- Hotel Listings
- Booking Management
- Admin Dashboard
- JWT Authentication
- MySQL Database Integration
- Booking History
- Cancel Booking Feature

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MySQL
- Authentication: JWT, bcrypt
