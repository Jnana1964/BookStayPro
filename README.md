# 🏨 BookStayPro – Hotel Management System

A Full Stack Hotel Booking and Management System developed using Node.js, Express.js, MySQL, HTML, CSS, and JavaScript.

---

# ✨ Features

## 👤 User Features
- User Login Authentication
- Explore Luxury Hotels
- Search Hotels
- Filter by City
- Sort by Price
- Book Hotels
- Booking History
- Cancel Booking
- Responsive Dark UI

---

## 🛠️ Admin Features
- Admin Dashboard
- View All Users
- View All Hotels
- View All Bookings
- Register New Hotels
- Manage Booking Data

---

# 🧠 Tech Stack

## Frontend
- HTML5
- CSS3
- JavaScript

## Backend
- Node.js
- Express.js

## Database
- MySQL

- JWT Authentication
- bcrypt Password Hashing

# 🚀 How to Run the Project

## Backend Setup

```bash
cd backend
npm install
npm run dev
```

Backend runs on:

```bash
http://localhost:3000
```

---

## Frontend Setup

Open frontend using Live Server.

Example:

```bash
http://127.0.0.1:5500/frontend/index.html
```
